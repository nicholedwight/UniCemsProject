Maya Planet

For my animation of a planet of my choice, I chose to stick with what I knew and make an animation of Earth.

I started off with a polygon sphere for the base of the planet.

Next I added on a lambert texture for the base material, it's a nice matte texture so the texture map won't be shiny when a light hits it.

After applying the lambert material, I added on the earth texture that was provided to us.

To make it look like an actual planet, lighting was added with a directional light, representing the Sun's light. Knowing next to nothing about degrees at which the sun's light hits the Earth, I just placed where I thought looked best.

Linear tangents were implemented to keep the Earth from starting off at a slower speed, and also to keep it from decreasing nearer the end of the rotation. This just kept it with a smooth rotation. I also had a small problem with my planet rotating unusually fast, this was solved by adjusting the full rotation, by adjusting the playback speed to real-time.

The final step in making the animation look like a planet is to add in the axis tilt. So I adjusted the Z axis tilt to 22.5 degrees and below is what it looks like in the viewport so it can be seen from all angles.

Scripting Exercise

Guessing Game

The final project is a 'Guessing Game' is the last project. I used the getRandomNumber function from a previous exercise and inserted it into the code for the game, it creates a random number, which I then stored as a global variable so there's no cheating! I found that if it wasn't a global variable, the user can hit enter over and over and eventually get it right and win. I then took that variable for the random number function and compared that to the user input. If they were equal, I set the output to displayNumberEntered.text = "It was " + randomNumber + ". You're right!". Using the same process, outputs were set for when the user is too high or low, if the user guesses correctly, and if the user enters an invalid number i.e less than 1 or greater than 10. At first I was making this project way more difficult than it needed to be, by trying to create a second function to process the user input for the second guessed number, the function itself worked, however, I couldn't figure out how to set the stage to the second half, after the first processNumberInput function. It then occured to me to set the random number variable globally and use the original input sections.
